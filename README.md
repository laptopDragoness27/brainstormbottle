Brainstorm in a Bottle: an application to help organise, sort, and decide upon ideas to help with brainstorming.

Requirements:

-be able to add items to a list of ideas

-be able to sort the items by dragging them with the cursor in a chosen order

-be able to save the lists you make

-be able to easily remove items that you added to your list

-be able to undo any action you perform
